SpeccyBoot
How to use it

In addition to a SpeccyBoot-equipped Spectrum, two more machines are needed in your network. However, your probably already have them:

  • A DHCP server (virtually all broadband routers include this)
  • A TFTP server (a server application on a desktop/laptop computer)

These should be on the same network. Typically, you will just plug in both your Spectrum and your TFTP server to your router.

NOTE: The current version of SpeccyBoot does not support more than one SpeccyBoot on the same local network (the MAC address is hard-coded).

TFTP server configuration

Your TFTP server must be capable of responding to broadcast requests, since that is what SpeccyBoot uses. Most (but not all) TFTP servers are, and some need an explicit option to enable this. I used the built-in TFTP server of Mac OS X 10.5 (here's some hints on how to enable that), and most Linux distributions either include a TFTP server by default, or allow you to install one quite easily. I believe there are some third-party TFTP servers for Windows too.

Preparing a directory of snapshots

Create a directory named speccyboot in your TFTP servers's main directory (in the case of Mac OS X 10.5, this is /private/tftpboot; many UNIX clones use /tftpboot). Copy your Z80 snapshots to this speccyboot directory, and create a (case-insensitively sorted) list of snapshots named snapshots.lst. On a UNIX system, this can be done as follows:

# sudo mkdir /private/tftpboot/speccyboot
# sudo chown <your-username> /private/tftpboot/speccyboot
# cd /private/tftpboot/speccyboot
# cp <path-to-snapshots>/*.z80 .
# ls -1 *.z80 | sort --ignore-case > snapshots.lst

A few notes on this:

  • It could be convenient to put the last line in a script, so you can update the list easily whenever the directory is modified.
  • The sorting is not strictly required — but without it, you will not be able to jump (predictably) in the snapshot list by pressing an alphabetic key.

Loading snapshots

If the steps above went well, you should now be able to turn on your Spectrum, see it figure out its IP address aswell as that of the TFTP server, and present your list of snapshots. You can now use the arrow keys to navigate the list, the alphabetic keys to jump directly to snapshots whose name begin with that letter, and Enter to load the selected snapshot.

To boot into BASIC rather than the SpeccyBoot menu, press Caps Shift and keep it pressed while power-cycling/resetting the Spectrum.
